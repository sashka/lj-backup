# lj-backup

Usage
-----

1. Edit `lib/lj_backup.py` to set your username and password:

    `main('lj_username', 'password', '.')`

2. Run `./lj_backup`

3. Open your username forder.

4. Enjoy.


Features
--------
1. Store everything in JSON.
2. Download comments for every post.
3. Render simple HTML pages.
